2. 取得する情報

Google ログインで取得する識別子、メールアドレス、表示名、利用者が入力・アップロード・作成する保育書類、園児情報、操作・フィードバック情報を取得します。

3. 利用目的

本人確認、アカウントとデータ領域の分離、書類作成・保存・出力、品質改善、お問い合わせ対応、不正利用防止に使用します。

4. 保存・第三者提供

データは Google Cloud 上のサービスを利用して処理・保存します。生成AI機能の実行時には入力内容を Google Cloud の Vertex AI に送信します。法令上必要な場合を除き、本人の同意なく第三者へ提供しません。

5. ユーザーごとの分離

書類、園児、クラス、フィードバックは Google アカウントごとの個人ワークスペースに分離します。他の利用者のデータは閲覧・操作できません。

6. 削除

ログイン後の「データ削除」から依頼できます。受付後30日を目安に、対象ワークスペースの書類・園児・クラス・フィードバック・アカウント情報を削除します。法令上の保存義務がある情報はその範囲で保持します。

7. 改定

本ポリシーは必要に応じて改定し、このページで公開します。
