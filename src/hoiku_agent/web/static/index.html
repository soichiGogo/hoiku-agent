① 提案の差分 / ② 競合チェック
修正差分から、どの見出しに何を足すかを提案します。競合があれば保育士が正を選びます。
③ 評価ゲート（3軸 / main 比）
指針整合・10の姿・保護者向け表現の3軸で採点。main 比で非劣化かつ must_fix 0 なら緑。
④ PR（取り込み）
緑なら PR を起票（既定 dry_run）。採否は CI 評価ゲートが決めます。
